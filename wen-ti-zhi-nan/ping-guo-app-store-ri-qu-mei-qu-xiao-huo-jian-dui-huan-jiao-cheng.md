# 苹果app store日区（美区） 小火箭兑换教程

小火箭购买地址

[https://id.wohaobang.cn/product/8.html](https://id.wohaobang.cn/product/8.html)

手动兑换方法：

1、手机App Store登录美区ID

2、点击Redeem Gift Card or Code

3、复制邮件里的兑换码

4、点击You can also enter your code manually

5、把兑换码粘贴进去

6、点击右上角Redeem即可下载
